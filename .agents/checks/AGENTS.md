## Repo-Local Checks

- Keep review, audit, and check workflows under `.agents/checks`.
- Store each check workflow as a direct `.md` file named `<check-name>.md`.
- Include `name` and `description` frontmatter in each check workflow file.
- Keep workflow Markdown concise and focused on agent-facing instructions.
- Update root `AGENTS.md` and `.agents/AGENTS.md` when check locations or validation rules change.
